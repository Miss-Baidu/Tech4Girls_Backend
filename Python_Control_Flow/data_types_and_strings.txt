
The int type is used for representing whole numbers, thtat is both positive and negative.
example of int type is
Height = 6
weight = -20

The float type represents numbers that have decimals or fractional parts.
Example of a float is 
Cost = 13.5
Temperature = -6 .5 

The str type is used for representing strings, which are sequences of characters
Example of a str is
Name = 'Sarah'
Greeting = 'Hello, I am Sarah'

A list is a mutable, ordered collection that can hold multiple items.
Example of a list is
Items = ['Rice', 'Oil', 'Onions', 'pepper']

A tuple is similar to a list but is immutable, meaning its contents cannot be changed after creation. And its represented by parantheses 
Example of a tuple is
Names = ('Ama', 'Akua', 'Adjoa',)

A set is an unordered collection of unique items. Its prevents duplicaton.
Example of a set is
Numbers = {2, 5, 8, 1}

UPPER
text = "hello, My name is Sarah"
print(text.upper()) ouput = HELLO, MY NAME IS SARAH 

LOWER
text = "I am a student"
print(text.lower())  ouput = i am a student

REPLACE
text = "I am a T4G Scholar"
new_text = text.replace("Scholar", "student")
print(new_text)  ouput = I am a T4G student

SPLIT
text = "Sarah is a girl"
words = text.split()
print(words)  # Output: ['Sarah', 'is', 'a', 'girl']

JOIN
words = ['Hi', 'I'm', 'Sarah']
sentence = ' '.join(words)
print(sentence)  ouput = Hi, I'm sarah

FIND
text = "Hello, World"
index = text.find("world")
print(index)  ouput =7
